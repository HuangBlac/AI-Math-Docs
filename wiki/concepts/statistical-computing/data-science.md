# 数据科学导论

数据科学是统计学、计算机科学与领域知识的交叉。

## 核心流程

1. **数据获取**: 爬取、API、数据库
2. **数据清洗**: 缺失值、异常值、格式统一
3. **探索性分析 (EDA)**: 可视化、描述统计
4. **建模**: [[machine-learning|机器学习基础]]
5. **评估与部署**: 模型验证、生产化

## 工具箱

- R / Python (pandas, numpy, scikit-learn)
- [[statistical-computing|统计计算]] 方法支撑建模和推断
- 可视化: ggplot2, matplotlib

## 与统计学习理论的关系

数据科学偏重实践，[[erm|经验风险最小化]] 和 [[rademacher-complexity|Rademacher 复杂度]]
为其中的模型选择提供了理论基础。
